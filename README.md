# 🛡️ The Vault — ZatiAfriVault

> Securing Uganda's Communication — One Message at a Time

## Overview
ZatiAfriVault is a secure encrypted communication platform designed 
for Ugandan businesses and organizations. It protects sensitive 
conversations from interception, data breaches and cyber attacks 
using military grade AES-256 encryption.

## Features
- 🔐 AES-256 End-to-End Encryption
- 👥 Real-time secure chat rooms
- 🏢 Business focused rooms — General, Finance, Legal, HR, Executives
- 🔑 Secure user authentication with bcrypt password hashing
- 📱 Clean professional interface
- 🇺🇬 Built for Ugandan businesses

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | Python Flask |
| Real-time | Flask-SocketIO |
| Database | SQLite + SQLAlchemy |
| Encryption | AES-256 via Cryptography |
| Password Hashing | Bcrypt |
| Frontend | HTML, CSS, JavaScript |
